One Page Website Template

A clean and minimal one-page scrolling website – powered by Bootstrap

How it works

Each section is essential a "slide" that displays the full width and height of the screen.

You can use the navigation to jump to each section – or you can simply scroll up or down.

Navigation is fixed to the top of the page so you can easily get to each section.

Powered by Bootstrap

This template is built on the Bootstrap framework - a popular framework used to build responsive and extendable websites.

Do more with a grid system

Extend your slide content even further by leveraging a flexbox grid system – built right into Bootstrap!

Each row can be divided into 12 columns (all columns are stacked on mobile displays).

You can create any combination you want, spanning multiple columns.

This row is an example of 3 pieces of content, divided evenly across 4 columns.

Change the background color of a row for an even more customized look!

Background image

With a little bit of CSS code, you can change the background of your slides to include an image. The possibilities of this template really are endless.

Let's sum it all up

Super responsive, clean, minimal one page website with a wide array of uses.

Semantic HTML 5 template – built on the Bootstrap framework for ease of use.

Multiple layout formats that can be customized to fit you needs.

Perfect for creating custom landing pages, personal website, presentations and more!

Can be modified using a basic HTML editor (like Brackets or Dreamweaver).

Perfect for the beginner designer – no super heavy programming or skills required.

Cross-browser compatibility – works on any web browser on any device.
